# Bugra Kaan Saglam - Portfolio

Personal portfolio website built with Flutter Web.

## Highlights
- TR/EN localization
- Responsive layout with reusable sections
- Project and privacy policy pages
- Centralized theme tokens and typography

## Tech Stack
- Flutter (web)
- go_router
- easy_localization
- google_fonts

## Getting Started
```bash
flutter pub get
flutter run -d chrome
```

## Build
```bash
flutter build web
```

## Project Structure
- lib/app - app shell, router, theme, dependencies
- lib/core - cross-cutting utilities
- lib/features/portfolio - data, domain, presentation
- assets/translations - i18n JSON
- assets/images - media assets
